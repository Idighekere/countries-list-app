## Overview
Countries Explorer is a professional mobile application built with **React Native** and **Expo** that provides comprehensive data on nations worldwide. Using the REST Countries API, the app delivers a seamless user experience for searching, filtering, and studying global demographics, geography, and cultural information.

## Features
- **Real-time Search**: Instantly find any country using the optimized search bar.
- **Advanced Filtering**: Categorize countries by continents and specific timezones through a custom accordion-style modal.
- **Dynamic Theming**: Fully integrated Dark and Light mode support with local persistence via AsyncStorage.
- **Rich Details**: View in-depth statistics including population, official languages, currencies, driving sides, and area measurements.
- **Alphabetical Grouping**: Organized list view with sticky headers for efficient navigation through hundreds of entries.

## Getting Started

### Installation
Follow these steps to set up the development environment on your local machine:

1. **Clone the Repository**
   ```bash
   git clone git@github.com:Idighekere/countries-list-app.git
   cd countries-list-app
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Server**
   ```bash
   npx expo start
   ```

### Running the App
- For **iOS**: Press `i` to open the simulator.
- For **Android**: Press `a` to open the emulator.
- For **Web**: Press `w` to open in your browser.

## Usage
The application is designed for intuitive navigation:
- **Home Screen**: Browse the alphabetically grouped list of countries. Use the search bar at the top to filter results by name.
- **Filter Modal**: Tap the filter icon to open the selection menu. You can select multiple continents or timezones to narrow your search.
- **Theme Toggle**: Use the sun/moon icon in the header to switch between light and dark visual modes.
- **Details Screen**: Click on any country card to view its full profile, including its flag and detailed regional data.
